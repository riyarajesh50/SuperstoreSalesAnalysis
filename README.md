# Superstore Sales Analysis Dashboard

## Project Overview
This project analyzes retail sales performance using the Superstore dataset.

## Tools Used
- Python
- SQL
- Power BI

## Key Insights

### Category Performance
- Technology generates highest profit
- Furniture profitability impacted by discounts

### Regional Trends
- West region leads sales
- South region underperforms

### Discount Impact
- Discounts above 20–30% reduce profit

### Customer Segments
- Consumer segment drives most profit

### Seasonal Patterns
- Sales peak during Nov–Dec holiday season

## Business Value
Insights support pricing optimization, inventory planning, and regional strategy.
